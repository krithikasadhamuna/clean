FRONTEND API LIST FOR CODEGREY AI SOC PLATFORM

BASE URL:
http://15.207.6.45:8080

READY APIs (NO DEPENDENCIES):

1. SOFTWARE DOWNLOAD API:
   GET /api/backend/software-download
   Returns: 3 OS downloads (Windows, Linux, macOS) with exact structure provided

2. AGENT LISTING API:
   GET /api/backend/agents
   Returns: 4 AI agents (PhantomStrike AI, GuardianAlpha AI, SentinelDeploy AI, ThreatMind AI)
   - First 2: Active
   - Last 2: Inactive (UI will disable by default)

3. NETWORK CANVAS/TOPOLOGY API:
   GET /api/backend/network-topology
   Returns: Network nodes and connections in tabular format

OPTIONAL QUERY PARAMETERS:

For Agents API:
GET /api/backend/agents?location=SOC Infrastructure
GET /api/backend/agents?agent_type=attack
GET /api/backend/agents?status=active
GET /api/backend/agents?sort_by=name&order=asc

For Network Topology API:
GET /api/backend/network-topology?hierarchy=desc
GET /api/backend/network-topology?hierarchy=asc
